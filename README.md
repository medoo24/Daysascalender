# Smart Calendar Builder

A fully offline, browser-based calendar table generator.

## Main features

- Choose any start date and any duration from 1 to 1000 days.
- Set any number of days per row.
- Automatically fit the calendar to A4, A3, Letter, or Legal paper.
- Use portrait or landscape orientation.
- Show dates as `17/7`, `17/07`, `17`, or a custom token-based format.
- Optionally show weekday names above each date.
- Add blank writing boxes, ruled lines, dot grids, or checklists.
- Control title, date, weekday, year, borders, radius, spacing, margins, and zoom.
- Select individual days, drag-select, or use Shift-click for ranges.
- Mass-select or color days by date range, weekday, month, every Nth day, or all days.
- Color-code months and optionally show a month legend instead of month names.
- Save presets in the browser.
- Export and import settings as JSON.
- Print directly to PDF using the browser print dialog.

## Run

Open `index.html` in any modern browser. No server or installation is required.

## Save as PDF

1. Click **Print / Save PDF**.
2. Choose **Save as PDF** in the browser print dialog.
3. Enable **Background graphics** for accurate color output.
4. Keep scale at 100% unless your browser preview indicates clipping.
